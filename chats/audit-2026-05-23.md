# Chat-to-artifact audit — 2026-05-23

A one-time audit answering: every artifact file that Claude produced inside a chat in `/chats/`, is it now on disk? If not, where is it in the chat and is it worth extracting?

## Method

For each chat archive, scanned every `tool_use` block of name `create_file`, `str_replace`, and `present_files`. Recorded the output `path`. Then matched each path against files currently in `/artifacts/`, `/drafts/`, and `/archive/source/`. Any artifact the chat produced but no file on disk represents = "missing".
